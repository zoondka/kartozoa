get /{src}/{info}.json
(srcInfo.jsonGet)
Returns information on a given source

Path parameters

src (required)
Path Parameter — Tile source name (see https://github.com/kartotherian/kartotherian-core#sources)
info (required)
Path Parameter — Either 'pbfinfo' for a pbf source or 'info' for an image source

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

get /{src}/{z}/{x}/{y}.{format}
(srcZXY.formatGet)
Returns a tile based on source, zoom, coordinates, and format. Possible values for the various parameters depend on the Kartotherian sources configuration (see https://github.com/kartotherian/kartotherian-core#sources)

Path parameters

src (required)
Path Parameter — Tile source name
z (required)
Path Parameter — Zoom level of the tile
x (required)
Path Parameter — X coordinate of the tile
y (required)
Path Parameter — Y coordinate of the tile
format (required)
Path Parameter — File format of the tile

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/octet-stream
  • application/json
  • image/png
  • image/jpeg
  • image/svg+xml
  • text/html

Responses

get /{src}/{z}/{x}/{y}@{scale}x.{format}
(srcZXY@scalex.formatGet)
Returns a tile based on source, zoom, coordinates, scale, and format. Possible values for the various parameters depend on the Kartotherian sources configuration (see https://github.com/kartotherian/kartotherian-core#sources)

Path parameters

src (required)
Path Parameter — Tile source name
z (required)
Path Parameter — Zoom level of the tile
x (required)
Path Parameter — X coordinate of the tile
y (required)
Path Parameter — Y coordinate of the tile
scale (required)
Path Parameter — Scale for rendering larger images to high resolution screens
format (required)
Path Parameter — File format of the tile
